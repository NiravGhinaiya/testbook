The React Context API is a powerful tool for sharing data across different components in a React application without the need for prop drilling.

Lazy loading is a design pattern that delays the loading of non-essential resources until they are needed. This technique is used to improve performance, reduce initial load time, and optimize resource usage in web applications.

Suspense is a built-in component in React that is used to handle the loading states of components or data. It allows developers to define a fallback UI that is displayed while the application is waiting for something, such as a lazy-loaded component or asynchronous data, to be ready.

memo is a higher-order component (HOC) that optimizes the rendering of functional components by preventing unnecessary re-renders. It does this by memorizing the component's rendered output and re-rendering it only when its props change.
